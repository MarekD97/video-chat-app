## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.

Installation:

`npm install`

To Run Client and Server:

`npm run dev`

To Start only Server:

`cd /server/`
`npm start`

To Start only Client:

`cd /client/`
`npm start`

To Visit App:

`localhost:8080`
